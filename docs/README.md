# docs/

GitHub Pages documentation and auto-generated map previews.

## Structure

```
docs/
  index.html        # GitHub Pages entry point (generated by CI)
  previews/         # Map preview images generated on each commit
```

## previews/

Map preview screenshots are generated automatically by the CI pipeline after each build and published here. They show how the current render theme looks for selected map areas.

Do not add files here manually — this directory is managed by the CI pipeline.
