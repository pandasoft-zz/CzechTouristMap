# maps/

This directory stores Mapsforge `.map` files used by the tile server.

## Czech Republic

Download `czech-republic.map` from OpenAndroMaps:

**https://www.openandromaps.org/en/downloads/europe**

Save it here as `maps/czech-republic.map`.

## Other regions

All available Mapsforge maps: **https://download.mapsforge.org/maps/v5/**
